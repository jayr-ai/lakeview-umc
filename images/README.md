# Images

Drop site images here.

- `umc-logo.svg` (or `.png`) - official UMC Cross and Flame, used in the nav and footer.
  Use a transparent background. SVG preferred; transparent PNG is fine. Not JPG.
- `umc-logo-white.svg` (optional) - reversed/white version for the dark footer.
- Ministry and hero photos can also live here (e.g. `hero.jpg`, `kids.jpg`).
